# Generative Art
These are my experiments in generative art using simple PHP, HTML and CSS. 

## Version 1 - 1st September 2021. 

### Rules: 
* Generate a bunch of squares. 
* Apply a colour to a square. 
* Pop an SVG blob onto a square. 
* Save as an output.
